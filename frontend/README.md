# ResumeAI Frontend

This directory contains the Next.js 16 frontend for the ResumeAI platform.

## 🚀 Quick Start

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) to view the app.
